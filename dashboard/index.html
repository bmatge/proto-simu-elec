Lecture

La France est sur la trajectoire de décarbonation côté production (l'intensité du kWh est déjà parmi les plus basses d'Europe), mais le rythme de déploiement des ENR, des VE et des PAC reste en retard sur les cibles intermédiaires. La production est gagnée, l'électrification des usages reste à accélérer.

Objectif éditorial

Tableau de bord public confrontant quatre angles de l'électrification française : (1) la décarbonation du mix électrique, (2) la bascule du parc roulant, (3) l'électrification du bâtiment, (4) la trajectoire vers les cibles 2050. Chaque onglet est autonome (lecture indépendante), mais partage un bandeau de KPI nationaux. Toutes les sources sont opendata et accessibles en API REST.

Stack technique

DSFR 1.14 dsfr-data web components dsfr-chart 2.0 ODS v2.1 data-fair API (ADEME) Etalab 2.0

Endpoints clés

Base ODRÉ : https://odre.opendatasoft.com/api/explore/v2.1/catalog/datasets/{slug}/records

Base ADEME DPE : https://data.ademe.fr/data-fair/api/v1/datasets/meg-83tjwtg8dyz4vv7h1dqe/values_agg?field={champ}&agg_size={N}&size=0

À retenir sur data-fair : utiliser agg_size pour le nombre de valeurs distinctes retournées (pas size) et size=0 pour ne pas embarquer les sous-résultats. sort=-count pour le tri par fréquence, sort=key pour l'ordre alphabétique (utile sur les étiquettes DPE A→G).

Trous identifiés

  • Prix de l'électricité ménages/industrie : non trouvé sur data.gouv → Eurostat nrg_pc_204 ou CRE
  • Stock annuel PAC installées : aucune série dédiée → reconstruction via DPE post-rénovation ou AFPAC
  • Bonus écologique / leasing social : pas en opendata → demander à l'ASP
  • Projets hydrogène : quasi inexistant → repli sur scénarios ADEME 2050 uniquement
